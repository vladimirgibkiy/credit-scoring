# credit-scoring
В этом проекте я создал с нуля консольную систему кредитного скоринга, которая помогает банку оценивать надежность заемщиков. Я использовал чистый язык Python без готовых Data Science библиотек.С помощью самостоятельно написанного алгоритма логистической регрессии и градиентного спуска я обучил модель находить скрытые зависимости между признаками клиентов (их возрастом, доходами, кредитной историей) и фактом возврата кредита. На выходе моя модель определяет точную вероятность дефолта для новых заемщиков, помогая автоматически принимать решения о выдаче или отказе в кредите.
